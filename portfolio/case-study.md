# DealLens Case Study

Status: draft placeholder

## Problem

Private equity teams receive CIMs and teasers in unstructured PDF form. First-pass review is manual, repetitive, and inconsistent.

## Solution

DealLens automates the first-pass CIM review workflow with Zapier, OpenAI, Airtable, Google Docs, and Slack.

## Architecture

```text
Google Drive -> Zapier -> PDF extraction -> OpenAI -> Airtable -> Google Docs -> Slack
```
