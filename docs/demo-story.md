# Demo Story

## One-Minute Pitch

DealLens automates the first-pass CIM review workflow for private equity teams. When a CIM is uploaded to Google Drive, Zapier extracts the document text, OpenAI converts it into structured deal intelligence, Airtable becomes the review workspace, Google Docs generates a first-pass memo, and Slack notifies the analyst team with key risks, missing fields, and review links.

## Manual Process

Before automation, an analyst manually reads a CIM, copies core company and financial fields into a tracker, drafts a first-pass memo, identifies risks, writes diligence questions, and sends a summary to the team.

## Automated Process

After automation, the CIM upload triggers extraction, structured data capture, memo drafting, risk flagging, diligence question creation, Slack notification, and impact tracking.

## Measured Impact

Target metrics:

- Manual review time per CIM
- Automated processing duration
- Estimated minutes saved
- Fields extracted
- Missing fields identified
- Risks created
- Diligence questions created
- Memo generation success rate
